## 1. 솔루션 개발 개요
본 솔루션은 고객사의 내부 보안 점검 상세 체크리스트 결과를 자동으로 가공하여, 자사 보안 솔루션 리스트 내에서 최적의 제품을 매핑하고, 법적 준거성 및 위험도 점수를 스케줄링하여 연차별 로드맵 엑셀 파일을 자동으로 작성해주는 **지능형 AX 자동화 솔루션**입니다.

- **개발 언어:** Python 3.13
- **주요 라이브러리:** `openpyxl` (서식 보존형 엑셀 파싱/라이팅), `pandas` (데이터 정규화 및 탐색), `google-generativeai` (Gemini API 기반 RAG 매핑), `python-dotenv` (환경변수 관리)
- **핵심 엔진 구성:**
  - **LLM 모드:** Gemini API와 RAG 기법을 조화시켜 정교한 CoT(Chain-of-Thought) 기반의 솔루션 매핑 및 구체적 비고(추천 사유)를 자동 기술
  - **Rule Fallback 모드:** 인터넷 비연결 혹은 API Key가 없는 환경에서도 정상 작동하도록 사전 정의된 150여 개의 자사 솔루션 키워드 매칭 논리로 로드맵 수립을 수행

---

## 2. 솔루션 사용 방법

에이전트는 CLI 기반 명령어로 간편하게 작동할 수 있습니다.

### 2.1 환경변수 설정 (API 모드 활성화 시)
1. 루트 디렉토리에 생성되어 있는 `.env.template` 파일을 복사하여 `.env` 파일을 생성합니다.
   ```bash
   cp .env.template .env
   ```
2. `.env` 파일 내 `GEMINI_API_KEY` 항목에 발급받은 Google Gemini API 키를 입력합니다.
   ```text
   GEMINI_API_KEY=AIzaSyA...
   ```
> [!NOTE]
> `.env` 파일이 없거나 API Key가 누락된 경우, 자동으로 **로컬 룰 기반 매퍼(Fallback Mode)**로 실행되어 중단 없이 안전하게 결과를 도출합니다.

### 2.2 파이프라인 가동 및 실행
터미널에서 아래의 파이썬 명령어를 실행하여 로드맵 매핑 및 작성을 시작합니다.
```bash

```
